# 📱 SnapVerse

SnapVerse is a full-stack social media platform inspired by Instagram, built using the MERN stack. Users can create posts, like, comment, follow other users, chat in real time, and manage their profiles through a modern and responsive interface.

## 🚀 Live Demo

Project Link :- https://social-frontend-production-2988.up.railway.app/

---

## ✨ Features

### Authentication
- Secure JWT Authentication
- Login & Signup
- Email Verification with OTP
- Protected Routes

### User Profile
- Edit Profile
- Upload Profile Picture
- Follow / Unfollow Users
- View Followers & Following

### Posts
- Create Post
- Edit Post
- Delete Post
- Like / Unlike
- Comment System
- Image Upload

### Real-Time Chat
- One-to-One Messaging
- Socket.IO Integration
- Instant Message Updates
- Chat History

### Feed
- Personalized Feed
- Responsive UI
- Infinite Scrolling (if implemented)

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/amansingh773/SnapVerse.git
cd SnapVerse
```

### Backend

```bash
cd social-backend
npm install
npm run dev
```

### Frontend

```bash
cd social-frontend
npm install
npm run dev
```
